# kill-more

## 安装教程

### 拉取代码

```sh
git clone git@github.com:onemue/kill-more.git
```

### 安装依赖

```sh
npm install
```

### 启动项目
```sh
npm run build-watch
# 随后在谷歌浏览器中插件中选择目录下的 dist 目录
```

### 打包项目
```sh
npm run build
```
